# Adaptive Math Quiz

A C# console application that generates math problems with dynamic difficulty based on user performance.

## Features
- 4 difficulty levels
- Random math problem generation
- Event-driven difficulty adjustment
- Delegate-based design
- Console-based user interaction

## Technologies
- C#
- .NET Console Application

## How It Works
The program increases difficulty after 2 correct answers in a row and decreases difficulty after 2 incorrect answers in a row.
